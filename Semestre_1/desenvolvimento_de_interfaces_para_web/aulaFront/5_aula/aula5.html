Introdução ao CSS

Existem 3 maneiras de vincular o css aos elementos HTML

CSS Externo:

Podems vincular qualquer arquivo HTML a um arquivo externo de CSS.

Essa é a forma mais correta de criar sites, principalmente onde há mais de uma página. pois o arquivo CSS fica separado do HTML e vinculamos o mesmo CSS a todas as páginas HTML.

CSS dentro do HTML:

Podemos adicionar estilos CSS diretamente no arquivo HTML, usando a tag STYLE.

CSS Inline(menos recomendável):

É possivel adicionar estilos CSS diretamente em cada tag, usando o atributo STYLE.
